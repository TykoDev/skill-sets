# Template 02: SaaS Product Page

**Scope**: Tailored for software products with interactive demos, feature
showcases, and integration logos. Extends Template 01 with SaaS-specific
sections.

**Layout Architecture**: Identical to Template 01 with additional sections.

---

## Additional Sections (beyond Template 01)

```
┌─────────────────────────────────────────────┐
│ INTEGRATION SECTION                          │
│ "Works with your favorite tools"             │
│ Grid of 8-12 integration logos/icons          │
├─────────────────────────────────────────────┤
│ HOW IT WORKS                                 │
│ 3-step horizontal process:                   │
│ [1] Connect → [2] Configure → [3] Deploy    │
├─────────────────────────────────────────────┤
│ INTERACTIVE DEMO SECTION                     │
│ Embeddable demo / video / animated mockup    │
│ "Try it live" CTA                            │
├─────────────────────────────────────────────┤
│ SECURITY & COMPLIANCE                        │
│ Badges: SOC2, GDPR, HIPAA (as applicable)    │
│ Brief security description                   │
├─────────────────────────────────────────────┤
│ COMPARISON TABLE (vs competitors)            │
│ Feature comparison grid with checkmarks      │
└─────────────────────────────────────────────┘
```

## Key Differences from Template 01

- Integration logos section emphasizes ecosystem compatibility
- "How It Works" reduces perceived complexity to 3 steps
- Interactive demo replaces static hero for software products
- Security/compliance builds enterprise trust
- Comparison table addresses competitive objections

## Responsive Behavior

Same as Template 01. Integration logos wrap 2×3 on mobile, 3×2 on tablet,
6-across on desktop. Comparison table scrolls horizontally on mobile.
