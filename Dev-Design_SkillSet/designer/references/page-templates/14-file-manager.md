## Key Features

- Drag-and-drop upload
- Grid/list view toggle
- File type icons
- Multi-select (Shift+click, Ctrl+click)
- Bulk actions: Download, Delete, Move, Copy
- Context menu: Rename, Move, Delete, Download, Share
- Breadcrumb navigation for folders
- File preview panel
