# Template 09: SaaS Pricing Page

**Scope**: Standalone pricing page for subscription products. Optimized for
plan comparison and conversion.

---

## Pricing Card Design

- Plan name (bold)
- Price (display size) + "/month" (muted)
- Annual savings callout when annual toggle active
- Description (1 line)
- Feature list with checkmarks/crosses
- CTA button (primary for recommended, outline for others)
- "Most popular" badge on recommended plan
