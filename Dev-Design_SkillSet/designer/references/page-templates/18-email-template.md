# Template 18: Email Template (Transactional)

**Scope**: Transactional emails (welcome, password reset, order confirmation).

---

## Section Breakdown

```
┌──────────────────────────────────────────────┐
│ HEADER: Logo + Company Name                  │
├──────────────────────────────────────────────┤
│ GREETING: "Hi [First Name],"                 │
├──────────────────────────────────────────────┤
│ PRIMARY CONTENT                              │
│ Heading: "Confirm your email address"        │
│ Body: "Click the button below..."            │
│ [████ Confirm Email Address ████]             │
│ Alternative: "If button doesn't work..."     │
│ with clickable URL                           │
├──────────────────────────────────────────────┤
│ SECONDARY CONTENT (optional)                  │
├──────────────────────────────────────────────┤
│ FOOTER                                       │
│ Company address + unsubscribe link           │
│ Social links + "You're receiving this..."    │
└──────────────────────────────────────────────┘
```

## Email Template Rules

- Table-based layout (no CSS Grid/Flexbox)
- Inline all CSS styles (no external stylesheets)
- Max-width: 600px
- System font stack (no web fonts)
- Alt text for all images
- Plain text version included
- Preheader text (hidden preview)
