# Template 11: Settings / Configuration Page

**Scope**: User and application settings with logical groups.

---

## Section Breakdown

```
┌──────────────────────────────────────────────┐
│ PAGE HEADER: Settings                        │
├──────────────────────────────────────────────┤
│ [Profile] [Account] [Notifications] [Theme]  │
├──────────────────────────────────────────────┤
│ ┌─── PROFILE ──────────────────────────────┐ │
│ │ Avatar upload + Name + Bio inputs        │ │
│ └──────────────────────────────────────────┘ │
│ ┌─── NOTIFICATIONS ────────────────────────┐ │
│ │ Toggle: Email / Push / Marketing         │ │
│ └──────────────────────────────────────────┘ │
│              [Save Changes]                   │
└──────────────────────────────────────────────┘
```

## Standards

- Group related settings with section headings and descriptions
- Toggle switches for booleans
- Select dropdowns for multi-option
- Radio buttons for mutually exclusive (≤3 options)
- Always show "Save" button (or auto-save with "Saved" confirmation)
- Provide "Reset to defaults" per group
