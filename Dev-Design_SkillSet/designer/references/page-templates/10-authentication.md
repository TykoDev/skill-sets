# Template 10: Authentication Pages

**Scope**: Login, registration, password reset, email verification.

---

## Centered Card Variant

```
┌──────────────────────────────────────────────┐
│         ┌──────────────────────┐             │
│         │  [Logo]              │             │
│         │  Sign in to [App]    │             │
│         │  [Email input]       │             │
│         │  [Password input]    │             │
│         │  [Forgot password?]  │             │
│         │  [Sign In button]    │             │
│         │  ── or continue ──   │             │
│         │  [Google] [GitHub]   │             │
│         │  Don't have an acc?  │             │
│         │  [Sign up]           │             │
│         └──────────────────────┘             │
└──────────────────────────────────────────────┘
```

## Split-Screen Variant (Desktop)

```
┌──────────────────┬──────────────────────────┐
│ BRAND PANEL      │ FORM PANEL                │
│ (60% width)      │ (40% width)               │
│ Brand colors     │ [Logo]                    │
│ Illustration     │ Sign in form              │
│ or Testimonial   │ Social sign-in            │
│                  │ Sign up link              │
└──────────────────┴──────────────────────────┘
```

## Responsive Behavior

- **Mobile**: Full-width centered card, minimal padding
- **Desktop**: Split-screen (brand + form) OR centered card with decorative
  background
